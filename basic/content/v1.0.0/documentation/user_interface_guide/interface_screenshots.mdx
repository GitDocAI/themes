## Accessing the UTMStack Web Interface

UTMStack provides a web-based Graphical User Interface (GUI). To access the login page, use HTTPS in front of your server name or IP. After installation, the default user is `admin` and the password is generated during installation, typically found in `/root/utmstack.yml`.

### Required Ports for Web Access

To access the UTMStack web interface, specific ports must be open:

*   **80/TCP**: UTMStack Web-based Graphical User Interface Redirector.
*   **443/TCP**: UTMStack Web-based Graphical User Interface.

## Log Explorer Module Interface

The Log Explorer Module is a central, dynamic, and highly interactive tool within UTMStack, providing an exhaustive view of an organization's log data. It is engineered for real-time visibility, immediate filtering, and nuanced analysis, essential for proactive monitoring and effective cybersecurity management.

### Log Explorer Panel

The Log Explorer interface is designed for seamless navigation and efficient functionality. Its centerpiece is the Data Grid.

![Log Explorer](https://mintcdn.com/utmstack/OEt3XKGepmb6ogAs/images/logexplorer/logexplorerpanel.png?fit=max&auto=format&n=OEt3XKGepmb6ogAs&q=85&s=0849568511d25b3b7b2f02349f6ca582)

### Key Interface Elements

The Log Explorer module provides visually intuitive representations of logs and includes the following sections for data analysis:

*   **Source**: For selecting data sources.
*   **Filters**: For configuring filters to refine log data.
*   **Data Grid**: An extensive view of log data based on defined datasource and query. It supports logical grouping of queries, opening multiple tabs, and saving search criteria.
*   **The Log**: Details related to individual log entries.
*   **Options**: Additional settings for log view.
*   **Personalize Fields**: Customizing displayed fields.
*   **Save the information**: Option to save queries or information.
*   **Save Query**: Specific functionality to save a configured query.

## Summary

The UTMStack interface is designed for comprehensive data visualization and analysis, with dedicated modules like Dashboards for customizable insights and Log Explorer for real-time, interactive log management.
