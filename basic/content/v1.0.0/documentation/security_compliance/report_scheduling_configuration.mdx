# Report Scheduling Configuration

The Compliance Reports Selection and Scheduling Module is an essential feature within the cybersecurity software platform, designed to optimize the management of compliance with industry-specific regulations. This module provides a comprehensive solution for customizing, automating, and monitoring regulatory compliance status, ensuring efficient management of requirements through an intuitive panel view and a step-by-step scheduling process.

## Compliance Calendar Panel View

The Compliance Calendar Panel View offers a clear and concise overview of each defined schedule. This view includes key details for quick review and management of compliance schedules, facilitating necessary adjustments or updates.

Key details displayed:
*   **Name**: Identifies the name of the program.
*   **Schedule**: Displays the defined time intervals for report delivery.
*   **Filters**: Shows the customized filters applied to the information included in the reports.
*   **Action**: Provides options to edit or delete the compliance schedule.

## Step-by-Step Report Scheduling Configuration

Configuring report scheduling involves selecting desired reports, applying filters, setting delivery times, and confirming to save your automated report settings.

### 1. Create a New Schedule

To begin, navigate to the Compliance Schedule module and click on the “New Schedule” button.

### 2. Report Selection

A list of available reports will be presented. From this list, you can select the specific reports you are interested in for further configuration.

![Report Selection](https://mintcdn.com/utmstack/-JweBhSZIV7FRzin/images/compliance/create_compliance_schedule.png?fit=max&auto=format&n=-JweBhSZIV7FRzin&q=85&s=2542f78dd9e6257309c1de9df7b3a50c)

### 3. Filter Application

Apply filters to customize the information included in the reports. Filters can be configured according to criteria already defined based on the dashboard associated with the selected report.

![Filter Application](https://mintcdn.com/utmstack/-JweBhSZIV7FRzin/images/compliance/schedule_filters.png?fit=max&auto=format&n=-JweBhSZIV7FRzin&q=85&s=462b7302a07bb7bb5e76fed468732c86)

### 4. Time Configuration

Schedule the delivery of reports at predefined time intervals. An intuitive interface is provided to select the delivery frequency, the day, and the specific time when the reports will be sent by e-mail.

![Time Configuration](https://mintcdn.com/utmstack/-JweBhSZIV7FRzin/images/compliance/schedule_time.png?fit=max&auto=format&n=-JweBhSZIV7FRzin&q=85&s=d323c91f3f60d4f0085e13c3f3e693fb)

**Note**: The minimum number of time sequences is defined for one daily delivery.

### 5. Confirmation and Saving

Once you have selected the reports, applied the filters, and configured the delivery time, you must confirm and save the configuration. This information will be stored for future automatic deliveries.

## Benefits and Capabilities

The Report Scheduling Configuration, as part of the Compliance Reports Selection and Scheduling Module, provides several benefits:

*   **Automation**: Automates the delivery of compliance reports, ensuring they are generated and sent at predefined intervals.
*   **Customization**: Allows for the creation of custom reports for audits or compliance checks and assessments.
*   **Continuous Monitoring**: Supports the creation of compliance dashboards for continuous monitoring of regulatory status.
*   **Regulatory Support**: Enables leveraging existing reports for various compliance standards such as HIPAA, GLBA, GPDR, SOC, FISMA, CMMC, and PCI-DSS.
*   **Simplified Compliance Management**: Simplifies compliance management by providing built-in reports and interactive dashboards, reinforced by an event and logs explorer for advanced analysis and a report/dashboard builder.

## Summary

The Report Scheduling Configuration documentation outlines the process for setting up automated compliance reporting within the platform. Users can select specific reports, apply custom filters based on dashboard criteria, and define delivery times and frequencies for email distribution. This module is crucial for organizations aiming to streamline compliance management, offering an intuitive interface for customizing, automating, and monitoring regulatory requirements, and ensuring timely access to essential compliance data.
